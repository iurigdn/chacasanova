INSTRUÇÕES DE USO

Sistema de Controle - Chá de Casa Nova

Visão Geral do Sistema

Parabéns! Seu sistema de controle para o Chá de Casa Nova está pronto para uso. Este sistema foi desenvolvido para ajudar você a gerenciar os números escolhidos pelos participantes do seu evento.

O sistema inclui:

  • Uma interface web interativa para visualização da cartela de números
  • Funcionalidade para seleção de números e registro de participantes
  • Armazenamento local dos dados dos participantes e números escolhidos
  • Página detalhada com instruções de pagamento para os participantes

Estrutura de Arquivos

O sistema é composto pelos seguintes arquivos:

cha_casa_nova/ ├── index.html # Página principal com a cartela e formulário ├── instrucoes_pagamento.html # Página com instruções detalhadas de pagamento ├── numeros.json # Arquivo de dados para armazenar o estado dos números ├── controle_numeros.py # Script Python para gerenciamento via linha de comando ├── selecao_direta.js # Script JavaScript para seleção direta na cartela └── todo.md # Lista de tarefas do projeto (pode ser removido)

Como Usar o Sistema

Opção 1: Interface Web (Recomendada)

A maneira mais fácil de usar o sistema é através da interface web:

  1. Abra o arquivo index.html em qualquer navegador web moderno
  2. A cartela de números será exibida, com os números disponíveis em verde e os escolhidos em rosa
  3. Para registrar um participante:
    • Clique diretamente no número desejado na cartela, ou
    • Use o formulário na seção "Adicionar Participante" para selecionar o número e preencher os dados
  4. Após adicionar um participante, o número será marcado como escolhido na cartela e o participante aparecerá na lista

Dica: Você pode compartilhar o link para a página de instruções de pagamento (instrucoes_pagamento.html) com os participantes para que eles saibam exatamente como proceder com o pagamento.

Opção 2: Script Python (Alternativa)

Se preferir uma interface de linha de comando, você pode usar o script Python:

  1. Abra um terminal ou prompt de comando
  2. Navegue até a pasta cha_casa_nova
  3. Execute o comando: python3 controle_numeros.py
  4. Siga as instruções no menu interativo para visualizar a cartela, adicionar participantes e gerenciar o sistema

Importante: O script Python e a interface web usam o mesmo arquivo de dados (numeros.json), então as alterações feitas em um serão refletidas no outro. Evite usar os dois simultaneamente para prevenir conflitos.

Hospedagem e Compartilhamento

Para compartilhar o sistema com outras pessoas ou acessá-lo de diferentes dispositivos, você tem algumas opções:

Opção 1: Compartilhamento Local

Você pode simplesmente copiar a pasta cha_casa_nova para um pendrive ou compartilhá-la via serviços de armazenamento em nuvem como Google Drive ou Dropbox.

Opção 2: Hospedagem Web

Para uma experiência mais profissional, você pode hospedar o sistema em um serviço de hospedagem web:

  • GitHub Pages (gratuito)
  • Netlify (gratuito)
  • Vercel (gratuito)
  • Qualquer serviço de hospedagem web que suporte HTML/CSS/JavaScript

Dica: Se optar pela hospedagem web, será necessário implementar um backend para armazenar os dados, pois o armazenamento local (localStorage) só funciona no mesmo dispositivo. Entre em contato se precisar de ajuda com essa implementação.

Backup e Segurança

É importante manter seus dados seguros:

  • Faça backups regulares do arquivo numeros.json, que contém todos os dados dos participantes e números escolhidos
  • Se estiver usando a interface web, os dados são armazenados no localStorage do navegador. Limpar os dados do navegador pode resultar na perda dessas informações
  • Considere exportar periodicamente a lista de participantes para um arquivo separado como backup adicional

Personalização

Você pode personalizar o sistema de acordo com suas necessidades:

  • Edite o arquivo index.html para alterar cores, textos ou layout
  • Modifique as instruções de pagamento em instrucoes_pagamento.html
  • Adicione seu logotipo ou imagens personalizadas
  • Ajuste o valor dos números ou a data do sorteio conforme necessário

Dica: Se não estiver familiarizado com HTML/CSS, faça apenas pequenas alterações nos textos, mantendo a estrutura original dos arquivos.

Suporte e Contato

Se precisar de ajuda com o sistema ou tiver dúvidas sobre seu funcionamento, não hesite em entrar em contato.
